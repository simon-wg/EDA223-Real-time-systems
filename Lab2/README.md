Every command is as defined in the Lab-PM

## Examples

### Clear history

F

### Toggle deadlines

d

### Change tone volume

Lower: u
Raise: i

### Mute volume

m

### Change tempo

Set to 200: 200t
Set to 60: 60t

### Change key

Set to -4: -4k
Set to 2: 2

### Play/pause

p

### Toggle conductor mode

c

### Tap tempo

Tap 4 times to change tempo
Hold for 2 seconds to reset to 120
